# `AuthorizationURL.setRedirectURI()`

Sets the `redirect_uri` parameter. 

## Definition

```ts
function setRedirectURI(redirectURI: string): void;
```

### Parameters

- `redirectURI`

## Example

```ts
url.setRedirectURI("https://example.com/login/callback");
```
